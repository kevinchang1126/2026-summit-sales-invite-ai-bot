# 流通年會問卷關鍵題項對應

本文件聚焦 `流通_年會問卷.docx` 中流通團隊自訂的三個關鍵題項：

- `Q1_活動後協助需求`
- `Q4_AI導入關鍵因素`
- `Q5_AI採用時程`

其中：

- `Q1` 用來判斷客戶是否願意被接觸與推進。
- `Q4` 是流通場最重要的商機切題，用來找出客戶最在意的投資目的與經營議題。
- `Q5` 用來判斷導入溫度與推進節奏。

另參考 `PPT/流通活動後推進攻略.pptx`，流通活動後推進的核心觀念是：

- 用問卷找高商機。
- 第 4 題是眾中之重。
- 從投資目的往下找經營議題、管理議題、單一場景與數智員工。
- 活動結束後是商機的開始，要用問卷切出 `L1 / L2` 高機會客戶。

## 題項速查

### Q1：活動後希望如何協助

題目：本次研討會結束後，希望我們如何協助您？可複選。

選項：

- 到府討論貴公司需求，並討論可能的協助
- 檢視貴公司目前作業流程狀況並討論可能之協助
- 針對貴公司資訊需求再進一步說明鼎新解決方案
- 其他

### Q4：評估或導入 AI 的主要關鍵因素

題目：請問促使貴公司評估或導入 AI 的主要關鍵因素為何？可複選。

選項：

- 因應市場或同業競爭
- 推動營收成長
- 提升營運效率與員工體驗
- 提升客戶體驗與服務效率
- 強化企業韌性
- 強化資安與資訊治理能力
- 企業永續與創新
- 其他

### Q5：公司採用 AI 的時程規劃

題目：您認為公司將採用 AI 的時程規劃。單選題。

選項：

- 尚未評估
- 預計半年內導入
- 預計一年內導入
- 預計二年內導入
- 已導入

## 流通推進攻略重點

### 第 4 題是流通場的主判斷題

依 `流通活動後推進攻略.pptx`，流通場要先從第 4 題判斷客戶的「投資目的」，再一路往下切：

- 投資目的
- 經營議題
- 管理議題
- 單一場景
- 數智員工 / Agent

換句話說，回訪不要先從產品開始，而是要先回答：

- 客戶想靠 AI 解決什麼經營問題？
- 這個問題比較偏「增單」、「增效」、「降損」、「控風險」，還是「強化治理」？
- 哪一個單一場景最容易成為第一個切入點？

### 流通活動後優先看的三個訊號

1. `Q1`：願不願意被接觸與深入討論。
2. `Q4`：為什麼現在要評估 AI。
3. `Q5`：這件事要多快發生。

## Q1 機會判斷

| Q1 選項 | 機會等級 | 判斷 | 建議動作 |
| --- | --- | --- | --- |
| 到府討論貴公司需求，並討論可能的協助 | 高機會 | 客戶願意直接進入需求交流，屬於最可推進訊號 | 3 個工作天內安排業務或顧問回訪 |
| 檢視貴公司目前作業流程狀況並討論可能之協助 | 高機會 | 客戶已接受進入流程盤點，適合顧問式切入 | 直接安排流程盤點會談 |
| 針對貴公司資訊需求再進一步說明鼎新解決方案 | 中高機會 | 對方案有興趣，但可能尚未明確到流程層 | 先寄對應場景資料，再安排短訪談 |
| 其他 | 待判斷 | 需看填寫內容與 Q4/Q5 | 依內容補判斷 |

## Q5 導入溫度判斷

| Q5 選項 | 溫度 | 判斷 | 建議節奏 |
| --- | --- | --- | --- |
| 已導入 | 高 | 客戶可能已做工具或專案，適合談擴大、整合或優化 | 優先安排顧問訪談，切 Scaling 或第二階段場景 |
| 預計半年內導入 | 高 | 有明確短期行動壓力，適合快速盤點場景與資源 | 1 週內安排需求確認 |
| 預計一年內導入 | 中高 | 有中期規劃，適合用案例與路線圖推進 | 先場景盤點，再安排下一步 |
| 預計二年內導入 | 中 | 有方向但未急，適合用案例、活動、試點概念培育 | 以教育與低壓力追蹤為主 |
| 尚未評估 | 低 | 仍在認知或觀望階段 | 不強推，先做教育與場景啟發 |

## 高機會定義

符合以下任一條件，可列為流通場 `L1 / L2` 高機會名單：

- `Q1_活動後協助需求` 選「到府討論貴公司需求，並討論可能的協助」。
- `Q1_活動後協助需求` 選「檢視貴公司目前作業流程狀況並討論可能之協助」。
- `Q1_活動後協助需求` 選「針對資訊需求再進一步說明解決方案」，且 `Q5_AI採用時程` 為「預計半年內導入」、「預計一年內導入」或「已導入」。
- `Q4_AI導入關鍵因素` 明確選到「推動營收成長」或「提升營運效率與員工體驗」，且 `Q5_AI採用時程` 為「預計半年內導入」、「預計一年內導入」或「已導入」。

## 推進優先級

| 優先級 | 條件 | 回訪策略 |
| --- | --- | --- |
| L1 立即推進 | Q1 願意到府討論或流程盤點，且 Q5 為半年內、1 年內或已導入 | 直接安排訪談，聚焦單一場景立案 |
| L2 積極推進 | Q1 想了解方案，且 Q4 動機明確、Q5 有時程 | 先寄對應資料，1 週內安排短訪談 |
| L3 案例培育 | Q4 動機明確，但 Q5 為二年內或尚未評估 | 用案例、攻略與活動升溫 |
| L4 長期培育 | Q1 不明確、Q5 尚未評估，且動機模糊 | 內容經營為主，不強推 |

## Q4 對應主軸表

### 1. 因應市場或同業競爭

判讀：

客戶通常在意市場變化、競品行動、通路壓力或數位化速度落差。

回訪主軸：

- 不能只談效率，要談競爭速度與通路反應速度。
- 可切「多元接單 Agent」或「預測備貨 Agent」。

建議切入：

「如果現在主要壓力來自市場與同業競爭，AI 的價值不只是省人，而是讓接單、備貨、通路反應更快。」

### 2. 推動營收成長

判讀：

這是流通推進攻略中特別重要的主軸，屬於高商機訊號。

回訪主軸：

- 從增單不增人切入。
- 看多元接單、通路佈局、會員經營、進入新通路等議題。

建議切入：

「如果 AI 投資目的是推動營收成長，我們會優先找能夠直接影響接單效率、通路拓展或轉單率的場景，而不是先從後台工具開始談。」

對應場景：

- 多元接單 Agent
- 預測備貨 Agent

### 3. 提升營運效率與員工體驗

判讀：

這是流通推進攻略另一個最容易發揮的主軸，也屬於高商機訊號。

回訪主軸：

- 先找高頻、費力、跨系統、跨通路的工作。
- 重點在減人坑、減重工、減等待。

建議切入：

「如果重點是提升營運效率與員工體驗，我們會先從高頻又費力的流程切入，例如接單、對帳、查核、補貨、跨通路資訊整合。」

對應場景：

- 多元接單 Agent
- 通路對帳 Agent

### 4. 提升客戶體驗與服務效率

判讀：

客戶通常關心回應速度、服務一致性、會員互動或前台服務品質。

回訪主軸：

- 從客戶接觸點切入，而不是只談內部流程。
- 可延伸到客服 Agent、接單 Agent、會員服務或門市應用。

建議切入：

「如果目前優先考量是客戶體驗，那 AI 更適合先放在前台高接觸場景，例如接單回應、客服、會員互動或門市服務。」

### 5. 強化企業韌性

判讀：

客戶通常在意供應波動、缺貨、壓貨、交期不穩、異常反應慢。

回訪主軸：

- 用備貨、庫存、供應鏈韌性切入。
- 著重預測能力與異常應變。

對應場景：

- 預測備貨 Agent
- 效期觀測 Agent

### 6. 強化資安與資訊治理能力

判讀：

這類客戶對 AI 不一定不感興趣，而是先擔心權限、資料、風險與治理。

回訪主軸：

- 先談治理與權限，再談場景。
- 不適合一開始推高自主 Agent。

建議切入：

「如果目前對 AI 最大顧慮是資安與資訊治理，那第一步應該是先確認資料權限、流程邊界與可控場景，而不是直接追求全自動。」

### 7. 企業永續與創新

判讀：

客戶通常會從品牌、創新、韌性、效率或 ESG 延伸思考 AI。

回訪主軸：

- 用中長期轉型與新營運模式切入。
- 可先找一個具代表性的場景做示範。

## 流通場景與 Q4 對應

| 流通單一場景 | 對應問題 | 適合對應的 Q4 動機 | 切入角度 |
| --- | --- | --- | --- |
| 多元接單 Agent | 接單慢且費力 | 推動營收成長、提升客戶體驗、提升營運效率 | 增單不增人、縮短接單處理時間 |
| 效期觀測 Agent | 庫存變損失 | 強化企業韌性、提升營運效率 | 降低報廢與損耗，提早處理效期風險 |
| 預測備貨 Agent | 壓貨或缺貨 | 推動營收成長、強化企業韌性 | 降低缺貨與壓貨，提升週轉效率 |
| 通路對帳 Agent | 對帳高頻且費力 | 提升營運效率與員工體驗、強化治理 | 降低人工作業與錯帳風險 |

## 建議回訪流程

1. 先看 `Q1`：客戶願不願意被接觸。
2. 再看 `Q5`：這件事是現在要做，還是先觀望。
3. 最後看 `Q4`：客戶為什麼要評估 AI。
4. 依 `Q4` 對應到流通單一場景，不要一開始就談完整平台。
5. 若屬 `L1 / L2`，目標是安排訪談、盤點流程或形成立案機會。

## 典型組合與建議切入

### A：Q1 願意到府討論 + Q4 推動營收成長 + Q5 半年內導入

判斷：

高機會，適合立即推進。

建議切入：

「如果目標是營收成長，而且半年內就想啟動，我們建議先從最容易看見成效的接單或備貨場景切入，確認是否能做到增單不增人。」

優先場景：

- 多元接單 Agent
- 預測備貨 Agent

### B：Q1 檢視流程狀況 + Q4 提升營運效率與員工體驗 + Q5 一年內導入

判斷：

高機會，適合顧問式盤點。

建議切入：

「如果主要目的是提升效率與員工體驗，我們可以先一起看哪個流程最費力、最高頻、最容易跨系統卡住，再選一個單一場景做試點。」

優先場景：

- 通路對帳 Agent
- 多元接單 Agent

### C：Q1 方案說明 + Q4 強化企業韌性 + Q5 二年內導入

判斷：

中機會，適合案例培育。

建議切入：

「如果目前還在中期評估，我們建議先看庫存、備貨與損耗管理相關案例，先理解 AI 如何幫忙降低缺貨與壓貨風險，再決定要不要往下一步走。」

優先場景：

- 預測備貨 Agent
- 效期觀測 Agent

### D：Q1 不明確 + Q4 強化資安與資訊治理能力 + Q5 尚未評估

判斷：

偏低溫，需要先做治理型培育。

建議切入：

「如果目前主要顧慮是資安與治理，那不需要急著談導入規模，反而應先看有哪些可控、低風險的 AI 應用方式，並先釐清權限與資料邊界。」

優先策略：

- 先寄治理觀念與案例摘要
- 保留後續活動或顧問交流機會
