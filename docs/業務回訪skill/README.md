# 年會回訪助手 Knowledge

本資料夾整理年會講者音檔與字幕內容，目標是把演講中的知識點轉成業務回訪時可使用的問卷對應、追問方向與話術素材。

目前第一版來源：

- `音檔/李義訓.srt`
- `音檔/黃昱凱.srt`
- `音檔/黃盈彰.srt`

## 使用方式

1. 先看客戶問卷中勾選或填寫的痛點。
2. 對照各講者檔案中的「問卷對應知識卡」。
3. 選出與客戶狀況最接近的卡片。
4. 使用「業務回訪切入」作為開場，再用「建議追問」確認客戶現況。
5. 若客戶有明確需求，再延伸到對應產品、顧問服務或下一步盤點。

## 關鍵問卷題項

正式問卷已放入 `製造_年會問卷.docx`。目前 `knowledge` 優先對應第 1 題與第 4 到 8 題，不追求所有題目都對應，而是抓出最能支撐業務回訪的關鍵訊號：

- `Q1_協助需求`：安排人員了解需求、對 AI 方案感興趣、報名線上活動、參加線下體驗、先不用。
- `Q4_AI應用程度`：尚未起步、局部嘗試、點狀應用、系統整合、全面賦能。
- `Q5_急迫領域`：生產與供應鏈、財務與行政核銷、研發與技術支援、經營管理與決策。
- `Q6_Agent期待成效`：高頻度、高知識、高經驗、負荷率。
- `Q7_AI轉型挑戰`：數據品質不佳或尚未整合、員工排斥或恐懼、缺 AI 人才、ROI 不明確。
- `Q8_投入意願`：已有預算、積極評估、觀望、暫無規劃。

完整對應表請見 [製造問卷_關鍵題項對應.md](/Users/yunhuang/drinmac/VSCODE/年會回訪助手/knowledge/製造問卷_關鍵題項對應.md)。
