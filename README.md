# 2026 Summit Sales Invite AI Bot

## 繁體中文

這是一個為 2026 企業高峰會（Summit）銷售團隊專屬打造的 AI 邀請說帖輔助機器人。此專案協助業務人員根據客戶的產業、角色以及溝通管道，自動生成客製化的活動邀請說帖，並且內建了「優秀說帖排行榜」與「投票機制」，讓團隊能夠互相學習與分享最佳實踐。

### 核心功能 (Features)
*   **🤖 AI 說帖生成**：根據設定的目標對象（例如：製造業、資訊主管、Email 管道），自動撰寫專業的邀請文案。
*   **🏆 優秀說帖排行榜**：檢視團隊成員產出的高品質說帖歷史紀錄。
*   **👍 說帖投票系統**：支援對喜歡的說帖按讚 (Like) 或是倒讚 (Dislike)，讓優質內容浮出水面。
*   **⚡ 全端無伺服器架構**：使用 Cloudflare Pages 部署前端與 API，並整合 Cloudflare D1 作為原生邊緣資料庫。

### 專案架構 (Architecture)
*   **前端 (Frontend)**：Vanilla HTML / JavaScript / CSS (`public/` 目錄)
*   **後端 API (Backend)**：Cloudflare Pages Functions (`functions/` 目錄)
*   **資料庫 (Database)**：Cloudflare D1 Serverless SQL Database
*   **AI 整合 (AI Integration)**：可透過 Cloudflare Workers 搭配外接 AI 模型 (如 Gemini 或 Anthropic Claude)。

### 快速開始 (Getting Started)

#### 1. 安裝依賴套件
```bash
npm install
```

#### 2. 設定環境變數
請複製專案根目錄的範例檔案，並命名為 `.dev.vars`，接著填入您專屬的 API Keys。
```bash
cp .dev.vars.example .dev.vars
```
*(注意：`.dev.vars` 已加入 `.gitignore`，請確保不要將您的真實密碼 commit 到版本控制系統中。)*

#### 3. 初始化本地資料庫
執行以下指令來建立本地的 SQLite D1 資料庫：
```bash
npm run db:init:local
```

#### 4. 啟動本地開發環境
啟動包含前端頁面與邊緣函式的本地端伺服器：
```bash
npm run dev
```

#### 5. 部署到 Cloudflare Pages
```bash
npm run deploy
```

## English

This is an AI-powered invitation bot tailored for the sales team of the 2026 Summit. It helps sales representatives automatically generate customized event invitation pitches based on the client's industry, role, and communication channel. The project also features an "Excellent Pitch Leaderboard" and a "Voting System," allowing the team to learn from and share best practices.

### Features
*   **🤖 AI Pitch Generation**: Automatically drafts professional invitation pitches based on specified target audiences (e.g., Manufacturing, CIO, Email channel).
*   **🏆 Pitch Leaderboard**: View the history of high-quality pitches generated by team members.
*   **👍 Voting System**: Supports liking or disliking pitches to surface the best content.
*   **⚡ Full-stack Serverless**: Deployed using Cloudflare Pages for both frontend and API routing, with Cloudflare D1 as the native edge database.

### Architecture
*   **Frontend**: Vanilla HTML / JavaScript / CSS (located in `public/`)
*   **Backend API**: Cloudflare Pages Functions (located in `functions/`)
*   **Database**: Cloudflare D1 Serverless SQL Database
*   **AI Integration**: Integrates via Cloudflare Workers with external AI models (like Gemini or Anthropic Claude).

### Getting Started

#### 1. Install Dependencies
```bash
npm install
```

#### 2. Setup Environment Variables
Copy the provided example file and rename it to `.dev.vars`, then fill in your API Keys.
```bash
cp .dev.vars.example .dev.vars
```
*(Note: `.dev.vars` is included in `.gitignore`. Make sure you never commit your real credentials to source control.)*

#### 3. Initialize Local Database
Run the following command to bootstrap the local SQLite D1 database:
```bash
npm run db:init:local
```

#### 4. Start Local Development Server
Start the local server containing both the frontend pages and edge functions:
```bash
npm run dev
```

#### 5. Deploy to Cloudflare Pages
```bash
npm run deploy
```
